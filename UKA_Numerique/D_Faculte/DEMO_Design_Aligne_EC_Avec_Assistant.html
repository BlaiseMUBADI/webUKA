Comment ça marche :
  1. Étape 1: Sélectionnez un PROFESSEUR dans le tableau de gauche (la ligne devient orange)
  2. Étape 2: Le tableau du milieu affiche les ASSISTANTS disponibles pour ce professeur
  3. Étape 3: Sélectionnez un ASSISTANT (ou choisissez "Sans assistant")
  4. Étape 4: Le tableau de droite affiche les ECs disponibles
  5. Étape 5: Cochez les ECs pour les assigner au duo Professeur + Assistant
Légende des statuts EC :
  • ✅ Coché = EC assigné à ce duo (Prof + Ass)
  • ⬜ Vide = EC disponible
  • ❌ Grisé = EC déjà pris par un autre professeur
